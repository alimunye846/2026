# Aaliyah's Grace – Luxury Vercel + Stripe Store

## What's included
- React + Vite storefront
- Luxury editorial homepage styling
- Product catalog for sensitive-skin beauty essentials
- Vercel serverless Stripe Checkout API
- Success and cancel pages

## Environment variables
Add these in Vercel:
- `STRIPE_SECRET_KEY`
- `PUBLIC_BASE_URL`

## Local development
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Upload the folder to GitHub or import it directly into Vercel.
2. Add the required environment variables.
3. Deploy.

## Notes
- Prices are in GBP.
- Replace the sample product images with your own product photography before launch.
- Test with Stripe test keys first, then switch to live keys.
